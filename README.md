# Inteligencja Obliczeniowa / Computational intelligence
Course included basics of Machine Learning: Data processing, data cleaning, missing data imputation, normalisation, classification, processing of various CSV files containing data, including the `iris dataset`, genetic algorithms, `OpenCV` for image processing, object detecion algorithms, working with `pandas`, `scikit-learn` and `tensorflow`. Building `convolutional` and `recurrent neural networks`
## Projects:
## 0. [Simple neural network distinguishing cats from dogs](https://github.com/dawid-kalinowski/Inteligencja-obliczeniowa/tree/main/proj0)
Built an image classification model using to distinguish between cats and dogs, implemented in TensorFlow/Keras with data preprocessing
## 1. [Convolutional neural network identifying musical genres](https://github.com/dawid-kalinowski/Inteligencja-obliczeniowa/tree/main/proj1)
Developed a music genre classification system using `Librosa` library for audio / waves processing including Mel-frequency cepstral coefficients and Fourier transformation. Built a CNN model in TensorFlow/Keras, with a Flask web app that analyzes songs from YouTube and predicts genre probabilities.
## 2. [Recurrent neural network identifying musical genres](https://github.com/dawid-kalinowski/Inteligencja-obliczeniowa/tree/main/proj2)
Built an LSTM recurrent neural network for music genre classification.
